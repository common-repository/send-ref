Add hidden fields to Contact Form 7 (or other contact forms) with all the parameters in the URL.

== Description ==

This plugin is built by [Scan WP](http://scanwp.net "Scan WP") Theme and plugin detector.
Have you ever sent traffic to your sites with parameters?
Such as http://mysite.com?ref=google&banner=big

With "Send Ref" you can add hidden fields in your contact form such as contact form 7, and they will be sent with the form.

= Send Ref support =
Any questions regarding the plugin will be answered and handeled here: <a href="http://scanwp.net/our-plugins">Scan WP Plugins</a>

== Installation ==

The best practice to install the plugin is:

1. Download the plugin
2. In your admin panel, go to Plugins -> Add New
3. Click "Upload Plugin" and choose the Zip file you downloaded
4. After the plugin is uploaded click "Activate plugin"

If you rather do things manually:

1. Upload send-ref' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
